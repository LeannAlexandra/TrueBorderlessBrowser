True Borderless Browser

True Borderless Browser is a project made to have a youtube video play on a transparent overlay on screen, much like a pop-out window in firefox.

Features

  • Opacity Controls 5% - 100%
  • Moveable, Controlable and Sizable
  • Click-through: work through and behind the window uninterupted
